# 商人ワゴン仕様

## 基本フロー
- 商人ワゴンを開くと、最初に「買う」「売る」「やめる」を表示する。
- 「買う」を選ぶと購入画面を表示する。
- 「売る」を選ぶと売却画面を表示する。

## 共通UI
- 購入画面タイトルは「商人ワゴン 購入」。
- 売却画面タイトルは「商人ワゴン 売却」。
- 左上に単独の「買う」「売る」テキストは表示しない。
- 詳細ウィンドウの位置とサイズはアイテム画面と同じにする。
- 所持金は詳細ウィンドウの左側に表示し、通常より少し大きいフォントで表示する。
- アイテムスロットは共通描画を使う。
- アイテム画像の下側に購入金額または売却金額を表示できる。
- 金額テキストは中央揃えで、黒い6px半透明アウトラインを付ける。
- 金額テキストの背景矩形は表示しない。
- アイテム右上に任意の数を「×n」で表示できる。
- 右上の数テキストにも、金額と同じ黒い6px半透明アウトラインを付ける。

## 購入画面
- 商品数ぶんだけスロットを表示し、空きスロットは作らない。
- 商品ごとに在庫数を持つ。
- 在庫数は暫定で商品生成時にランダムな1～5個とする。
- 商品を1個購入するたびに、その商品の在庫数を1減らす。
- 在庫数が0になった商品は品切れ扱いにする。
- 商品右上に在庫数を「×n」で表示する。
- 購入できない商品はアイテム自体を半透明で描写する。
- 購入できない商品の価格数値は赤色で表示する。
- 「購入可否」「購入可能」は表示しない。
- 決定時は即購入せず、「買う」コマンドを表示する。
- 「買う」コマンドを決定すると購入する。
- 高額商品、珍品、限定品などの確認ダイアログは表示しない。

## 売却画面
- アイテム画面をベースに、リュックの8×3スロットをすべて表示する。
- 空きスロットも表示する。
- アイテム画像の下側に売値を表示する。
- 売却不可条件は現時点では保護中のみとする。
- 保護中のアイテムは売値部分に「保護中」と表示する。
- 売れないものを扱う場合は売値部分に「売れない」と表示する。
- 保護中または売れないアイテムは半透明で描写する。
- 「売却可否」「売却可能」は表示しない。
- スタック品のコマンドは「1個売る」「すべて売る」を表示する。
- スタック品でないもののコマンドは「売る」を表示する。
- 高価な個体品や強化済み品の確認ダイアログは表示しない。
